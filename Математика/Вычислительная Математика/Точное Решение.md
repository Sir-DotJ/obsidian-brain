# Точное Решение:
Алгебраически выведенный результат нахождения корня уравнения/системы уравнений.
У точного решения не может быть [[Допустимая Числовая Погрешность|погрешности]].
Обозначается $x^*$.

## Пример:
Точное решение уравнения $x^2 - x = -2$:     $x = 2$.
