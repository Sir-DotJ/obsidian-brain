## Определение:
[[Численное Дифференцирование#Методы решения задачи ДС|Метод]] решения задачи о [[Численное Дифференцирование|численном дифференцировании]] [[Дифференциальное Уравнение 2-го Порядка|ДУ 2-го порядка]] с двумя переменными. При этом **метод сеток** можно использовать для всех видов краевых задач.
Метод основан на применении [[Разностные Аппроксимации Второй Производной|разностной аппроксимации для второй производной]] для раскрытия тех же производных внутри предоставленного [[Оператор Лапласа|оператора Лапласа]], с последующим решением [[Системы Линейных Алгебраических Уравнений (СЛАУ)|СЛАУ]] любым доступным [[Методы Решения СЛАУ|методом]].

## Теория метода:
Из формулы выше: $$-\Delta u_{ij} + ku_{ij} = f(x,y)$$Где: $\Delta u(x,y) = \frac{\delta^2 u}{\delta x^2} + \frac{\delta^2 u}{\delta y^2}$ - [[Оператор Лапласа|оператор Лапласа]]; мы можем получить [[Системы Линейных Алгебраических Уравнений (СЛАУ)|СЛАУ]] подставив формулы [[Разностные Аппроксимации Второй Производной|разностной аппроксимации для второй производной]] вместо соответствующих производных, из чего мы получаем например уравнение вида: $$[\frac{u_{i+1,j} - 2u_{ij} + u_{i-1,j}}{h^2_x} + \frac{u_{i,j+1} - 2u_{ij} + u_{i,j-1}}{h^2_y}]+ku_{ij} = f(x_i,y_j)$$Где: $i=1,...,N_{x-1}$ и $j=1,...,N_{y-1}$. Для получения которого был использован [[Правый Разностный Аналог Второй Производной|правый разностный аналог для второй производной]] при раскрытии обеих вторых производных.
Погрешность такой замены выводится с помощью сравнения погрешностей всех использованных аналогов: самая плохая погрешность и будет являться погрешностью **метода сеток** (не учитывая погрешность при раскрытии производных краевых задач $2$ и $3$).
В данной ситуации использовали только один разностный аналог, погрешность которого: $O(h^2)$, из-за чего погрешность данного метода тоже будет $O(h^2)$.

Совместив формулу выше и предоставленные граничные условия (из данного типа краевой задачи), мы получаем решаемую [[Системы Линейных Алгебраических Уравнений (СЛАУ)|СЛАУ]]: 
$$\Bigg\{\,\begin{array}{rcl}
	[\frac{u_{i+1,j} - 2u_{ij} + u_{i-1,j}}{h^2_x} + \frac{u_{i,j+1} - 2u_{ij} + u_{i,j-1}}{h^2_y}]+ku_{ij} = f(x_i,y_j), i=1,...,N_{x-1},j=1,...,N_{y-1} \\
	u_{ij} = \varphi_{ij}:\;\;\,\Bigg\{\begin{array}{rcl}
	i=1,&j = 0,...,N_{y}-1 \\
	i=N_x,&j = 1,...,N_{y}-1 \\
	j=0,&i = 0,...,N_{x} \\
	j=N_y,&i = 0,...,N_{x} \\
\end{array}\Bigg\}
\end{array}\,\Bigg\}$$
Важно заметить, что в зависимости от примененного [[Методы Решения СЛАУ|метода]] решения [[Системы Линейных Алгебраических Уравнений (СЛАУ)|СЛАУ]], погрешность **метода сетки** может измениться в случае, если примененный метод имеет погрешность хуже, чем примененные аналоги производных.
Чтобы этого избежать, стоит применять [[Точные Методы Решения СЛАУ|точные методы решения СЛАУ]].

Также важно заметить, что в случае второй и третьей краевых задач, погрешность примененного разностного аналога для получения значения функции из производной также влияет на погрешность **метода сеток**.

После решения [[Системы Линейных Алгебраических Уравнений (СЛАУ)|СЛАУ]], для достижения необходимой [[Допустимая Числовая Погрешность|точности]] $\epsilon$ [[Шаг Сетки|шаги]] $h_x$ и $h_y$ обеих [[Равномерная Сетка|равномерных сеток]] уменьшаются в $2$ раза, после чего [[Приближенное Решение|приближенное решение]] [[Системы Линейных Алгебраических Уравнений (СЛАУ)|СЛАУ]] вычисляется снова. После чего результат сравнивается с предыдущим с помощью [[Допустимая Числовая Погрешность|точности]] $\epsilon$.
Повторяя этот процесс с помощью [[Метод Двойного Пересчёта|метода двойного пересчёта]] мы получаем [[Приближенное Решение|приближенное решение]] с [[Допустимая Числовая Погрешность|точностью]] $\epsilon$.

## Краткий алгоритм:
### Краткая постановка задачи:
Необходимо найти вектор [[Приближенное Решение|приближенных значений]] $u_h$ [[Точное Решение|точной функции]] $\varphi(x, y)$ для [[Дифференциальное Уравнение 2-го Порядка|ДУ 2-го порядка]] с двумя переменными с [[Допустимая Числовая Погрешность|точностью]] $\epsilon$ (Определенная краевая задача [[Дифференциальное Уравнение 2-го Порядка|ДУ 2-го порядка]] с двумя переменными).

### Критерии использования метода:
Функция $\varphi(x)$ такая, что:
- $\varphi(x)$ - непрерывная;
- $\varphi(x)$ - непрерывно дифференцируема.

### Ход алгоритма:
Шаги $4$ и $6$ необязательны для вычисления результата, они нужны только если мы разрабатываем новый метод.
1) Выбрали изначальные [[Шаг Сетки|шаги]] [[Равномерная Сетка|равномерных сеток]] $h_x, h_y$;
2) Выбрали способ приближенного вычисления производных с помощью [[Разностные Аппроксимации Второй Производной|разностных аппроксимаций для вторых производных]];
	1) Для примера выше получили раскрытие формулы в виде: $$[\frac{u_{i+1,j} - 2u_{ij} + u_{i-1,j}}{h^2_x} + \frac{u_{i,j+1} - 2u_{ij} + u_{i,j-1}}{h^2_y}]+ku_{ij} = f(x_i,y_j)$$ используя [[Правый Разностный Аналог Второй Производной|правый разностный аналог второй производной]] для обеих переменных.
3) Если дана вторая либо третья краевая задача, то приблизительно вычисляем значение функции из соответствующих производных, и запоминаем погрешность применяемого аналога;
4) Вычисляем погрешность **метода сеток** (либо если уже известно запоминаем его), при этом учитывая погрешность приближенного преобразования граничных условий:
	1) Для произвольной формулы выбираем самую плохую погрешность из применимых разностных аналогов для функции и граничных условий.
5) Используя раскрытую выше формулу и граничные условия, применяем [[Методы Решения СЛАУ|методы]] решения [[Системы Линейных Алгебраических Уравнений (СЛАУ)|СЛАУ]] и получаем вектор [[Приближенное Решение|приближенного решения]] $u_h$;
6) Учитываем погрешность используемого на шаге $(5)$ метода, чтобы вычислить общую погрешность **метода сеток**;
7) Уменьшаем шаги обеих равномерных сеток $h_x$ и $h_y$ в $2$ раза;
8) Используя раскрытую выше формулу и граничные условия, применяем [[Методы Решения СЛАУ|методы]] решения [[Системы Линейных Алгебраических Уравнений (СЛАУ)|СЛАУ]] и получаем вектор [[Приближенное Решение|приближенного решения]] $u_\frac{h}{2}$;
9) Сравниваем $u_h$ и $u_{\frac{h}{2}}$ по правилу $max_{1\leq i \leq N}{|u_{hi} - u_{\frac{h}{2}2i}|}\leq \epsilon$:
	1) Если неравенство выше ложно: $u_h = u_{\frac{h}{2}}$ и переходим к шагу $(7)$;
	2) Иначе, вектор $u_{\frac{h}{2}}$ и будет [[Приближенное Решение|приближенным решением]] с [[Допустимая Числовая Погрешность|точностью]] $\epsilon$.
